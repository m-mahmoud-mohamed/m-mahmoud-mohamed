# Intro Banner

This is the source for the animated intro banner (particles.js + typed.js),
inspired by [Raymo111's profile](https://github.com/Raymo111/Raymo111).

## How to create the GIF

1. Open `index.html` in your browser (or paste into [CodeSandbox](https://codesandbox.io))
2. Use [ScreenToGif](https://www.screentogif.com/) (Windows) or [Kap](https://getkap.co/) (Mac) to record ~8 seconds
3. Export as GIF, optimize at [ezgif.com](https://ezgif.com/optimize)
4. Save the file as `intro.gif` and upload it to this repo root
5. Add to README.md:

```markdown
<div align="center">
  <img src="https://raw.githubusercontent.com/m-mahmoud-mohamed/m-mahmoud-mohamed/main/intro.gif" width="100%"/>
</div>
```

## Preview locally

Just open `index.html` directly in any browser — no build step needed.
